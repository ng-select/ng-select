By default dropdown panel is rendered as child element of ng-select, but you can append dropdown to any element using appendTo.

If you place ng-select inside container with fixed height and hidden overflow then dropdown will not be fully visible.


It can be fixed by appending dropdown to body or other parent element. Use appendTo="body" only if:

  • The Parent has overflow: hidden or overflow: auto: If the select is inside a small div (like a sidebar or a table cell) and the dropdown gets cut off.
  • The Parent has z-index issues: If the dropdown is appearing behind another element (like a sticky header) and you can't fix it with CSS.
  • Inside a Modal/Dialog: Modals often have strict boundaries that clip child elements.
